## Badge de notification sur l'icône iOS

L'application supporte maintenant les badges de notification sur l'icône iOS/Android.

### Pour que ça fonctionne :

1. **Créer les icônes** (optionnel mais recommandé) :
   - `icon-192.png` : 192x192 pixels
   - `icon-512.png` : 512x512 pixels
   - Vous pouvez utiliser un générateur d'icônes PWA en ligne

2. **Ajouter l'application à l'écran d'accueil iOS** :
   - Ouvrir Safari sur iPhone
   - Aller sur l'application
   - Cliquer sur le bouton "Partager"
   - Sélectionner "Sur l'écran d'accueil"
   - L'icône apparaîtra avec le badge de notification

3. **Le badge s'affiche automatiquement** :
   - Le nombre de bons en attente apparaît sur l'icône
   - Mise à jour automatique toutes les 30 secondes
   - Compatible iOS 16.4+ et Android
